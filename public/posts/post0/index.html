À propos du site

Ceci est le premier article écrit pour ce site. Le site utilise Hugo avec le thème ananke lien. Le thème va sûrement évoluer à l’avenir.

  • L’écriture se fait en Markdown –> trouver un bon éditeur Markdown
  • github/git –> liste des commandes utiles, générer les clés SSH pour push vers github

Lorsque le site est lancé avec la commande hugo server -D, la modification puis la sauvegarde d’un fichier entraîne la mise à jour automatique du site.
